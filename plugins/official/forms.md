## 表单 {#forms}

### 基础样式 {#base-styles}

- `input[type='text']`
- `input[type='password']`
- `input[type='email']`
- `input[type='number']`
- `input[type='url']`
- `input[type='date']`
- `input[type='datetime-local']`
- `input[type='month']`
- `input[type='week']`
- `input[type='time']`
- `input[type='search']`
- `input[type='tel']`
- `input[type='checkbox']`
- `input[type='radio']`
- `select`
- `select[multiple]`
- `textarea`

### 用法 {#usage}

```js
// windi.config.js
export default {
  theme: {
    // ...
  },
  plugins: [
    require('windicss/plugin/forms'),
    // ...
  ],
}
```
